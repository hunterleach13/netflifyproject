Let's get started with some of the basics when starting your GitHub website!

  1. When getting started with GitHub first you must make yourself a Github account!
  2. Next you have to create you first repository and add/commit your first README.md file.
  3. After creating your first repository you will be propted with the steps of how to add and commit your files into GitHub
  4. First, type cd and drag the folder in which you are using to store your files, then click enter (you can also type in git status to see if you're inside your folder)
  5. Next, type in the command git init which will inticate you with a message saying "Initialized empty Git repository"
  6. Your next command will be git add README.MD (or you can just type in "git add .") you can then run "git status" and if your file shows up as green in the terminal your file ahs been added to your repo.
  7. Then you must comit your changes using the git commit -am and add a message describing the change you made, for example "My First Commit"
  8. Next, use the command git branch -M main in order to cahnge the master branch over to the main branch
  9. Then you must add the remote repo (located on GitHub's website) as the origin of this branch. This let's us send our commits to GitHub's webiste once we're done with all of our coding. The command is git remote add origin (and the link given to you by GitHub, simply copy and pasting this will be easier.)
  10. Finally, once you have committed your changes, you need to push your chamges to GitHub. Using "git push origin main"
